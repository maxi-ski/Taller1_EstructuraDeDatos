## Taller 1 Estructura de datos | "MusicStream"

"Este desarrollo corresponde al Taller 1 de la asignatura Estructuras de Datos en la UCN. El objetivo principal es la construcción de un reproductor de audio integral utilizando C++ como lenguaje base. Siguiendo los requisitos académicos, la gestión de datos se realiza exclusivamente mediante implementaciones manuales de memoria dinámica y punteros
